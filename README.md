# Movie Watchlist
[Live Demo](https://movie-watchlist-jte.netlify.app/)

Search the [Open Movie Database](https://www.omdbapi.com/) and add movies to your personal watch list!

## Features
- Search movies and view their name, IMDB rating, runtime, genres, and plot
- Add movies to watchlist stored on-device
- Remove movies from your watchlist
- Responsive UI for mobile and desktop, with effects like hovers, hiding films you already added, and more!
